# 슬라이싱(Slicing)

[:]를 통해 슬라이싱 가능

시퀀스를 특정 단위로 슬라이싱

<img src="photo" alt="image-20210719141626079" style="zoom: 50%;" />

```bash
[a,b,c,d,e][2:3]
#[c]
[a,b,c,d,e][1:4]
#[b,c,d]
```

<img src="photo\image-20210719141734424.png" alt="image-20210719141734424" style="zoom:50%;" />

포함, 미포함보다는 **인덱스를 기준으로 자른**다고 생각하면 간단!
