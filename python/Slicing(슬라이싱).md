# 슬라이싱(Slicing)

[:]를 통해 슬라이싱 가능

시퀀스를 특정 단위로 슬라이싱

<img src="4.Container(컨테이너).assets/image-20210719141626079.png" alt="image-20210719141626079" style="zoom: 50%;" />

```bash
[a,b,c,d,e][2:3]
#[c]
[a,b,c,d,e][1:4]
#[b,c,d]
```
