# Module (모듈)

## 모듈과 패키지

- 모듈
  - 특정 기능을 **파이썬 파일(.py) 단위로 작성**한 것
- 패키지
  - 특정 기능과 관련된 **여러 모듈의 집합**
  - 패키지 안에는 또 다른 서프 패키지를 포함

### 파이썬 표준 라이브러리 (PSL, Python Standard Library)

- 파이썬에 기본적으로 설치된 모듈과 내장 함수
  - 예시) random.py

- https://docs.python.org/3/library/
